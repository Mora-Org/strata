The signature primitive. An 80px column between sidebar and main, materialized only when the session has ≥1 fork — there is no chrome cost when the user hasn't branched yet. Each branch is a vertical bar 7px wide, color drawn from the --bloom-N ramp by branch index. Saturation = recency: the just-forked branch is fully saturated; the abandoned middle branch is faded back to opacity 0.48; the origin is older still. Horizontal position = depth: origin in column 1, first fork in column 2, deeper forks step right. The active branch has a small ▶ caret next to its top in the current accent. Junctions are 6×6 hairline L-arcs — not git-graph circles. Hover spawns a tooltip with branch name and message count; the tooltip spills out of the 80px column into the main canvas. The minimal case (left) shows what the tree looks like after the first fork; the full case (right) shows abandoned + recent siblings stacked in the same depth column.
