Hover-only affordance. Each message carries a fork action at its lower-right — but it lives at opacity: 0 with a 200ms reveal delay, so the reading column stays uncluttered. The visible state in this preview is the post-hover state (achieved here with a .shown debug class). The glyph is ⤴ (Unicode "rightwards arrow over leftwards arrow") and the shortcut is ⌥+F. No background, no border — just mono text in --fg-3, brightening to --fg-1 on hover-over-the-action itself. Click creates a new branch and triggers an inline "Forkado de "…" — branch <name>" editorial confirmation (the same pattern as banner-success, with a Bloom-2 teal left-rail).
