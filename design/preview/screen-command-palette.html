Two typefaces, one row. The command name is mono Geist Mono in --fg-1; the description that sits beside it is italic Fraunces (opsz 36, SOFT 40) in --fg-2. The typographic contrast IS the design — you read the commands as code and the descriptions as prose, both in the same row. Filtering highlights the matched substring with a small --accent-tinted background pill, not by darkening the rest of the name. The no-match state is just one centered italic line — no illustration, no "no results found" panel. Frame is .frame.marks (560px wide, ~22vh from top), scrim is 70% bedrock — enough to dim the conversation behind without blacking it out. Anti-patterns explicitly avoided: no command icons, no "recently used" section, no pinned commands, no auto-execute on single match.
